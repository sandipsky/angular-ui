Menu

A lightweight dropdown / popover. Project the trigger via [dropdown-display] and the panel via [dropdown-item] / [dropdown-content]. The panel renders in a fixed layer anchored to the trigger, so it escapes any overflow clipping, flips above the trigger when there isn't room below, and stays anchored while the page scrolls — the same behavior as the l-select dropdown. Scroll this page so a trigger sits near the bottom edge and the panel opens upward.
