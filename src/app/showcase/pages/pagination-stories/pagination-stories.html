Pagination

Page navigation with first / prev / next / last controls, a windowed page-number strip, and a page-size dropdown built on l-menu. Bind [length], optionally [pageSizeOptions], and react to (pageChange). Because the page-size dropdown uses the menu, it flips upward when the paginator sits near the bottom of the viewport.
