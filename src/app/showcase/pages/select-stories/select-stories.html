A single-select dropdown inspired by Ant Design. Accepts a plain array of values or an array of objects mapped via bindValue/bindLabel, with optional in-place searchable filtering, groupBy headers, an async-friendly loading state, and a virtualScroll mode for large datasets. Per-option disabling is driven by bindDisabled. Turn on multiple for tag-based multi-select with maxCount, maxTagCount and clearable.
