OTP / PIN Input

A row of single-character boxes for one-time codes and PINs, in the spirit of Ant Design's Input.OTP and Mantine's PinInput. Typing advances to the next box, Backspace walks back, arrow keys move between boxes, and pasting a code distributes it across them. Set [mask]="true" to hide the entered characters, type="number" to restrict to digits, and bind with [(ngModel)] or a reactive form. Sizes sm / md / lg, error state, and an optional separator are built in.
