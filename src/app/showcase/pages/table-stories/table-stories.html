Table

A data table styled from the app's _table.scss, with click-to-sort column headers that cycle ascending → descending → unsorted. Cells render row[key] by default, or a custom <ng-template lTableCell="key">. Sort locally (the table reorders data) or let the server sort ([serverSort]="true" — the table only emits (sortChange) and you pass sorted data back). sort is a two-way model.
