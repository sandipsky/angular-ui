Tree View

A hierarchical tree with Reddit-style connecting rails, inspired by Ant Design's Tree. Data-driven via [nodes]; a chevron expands/collapses each branch, [checkable] adds cascading checkboxes (with indeterminate parents), [showLine] toggles the rails, and [showIcon] renders per-node glyphs. Expanded / checked / selected state are two-way models, and expandAll() / collapseAll() can be driven from external buttons via a template ref.
